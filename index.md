# Niblio

**Niblio** is a flashcard app built by **Happi Snappi LLC** that helps you learn and retain anything using spaced repetition.

- 6 study modes — flashcards, quizzes, typing, true/false, match, and Smart Mix
- Science-backed spacing so nothing slips away
- Streaks, XP, and achievements to keep you motivated
- Import from Anki or create your own decks
